\subsection{Descripción detallada}
Clase \doxyref{Gestor}{p.}{classGestor}, gestiona los datos a traves de una lista enlazada y un \char`\"{}selector de idioma\char`\"{}. Sus atributos son:\begin{itemize}
\item lista: listaenlazada que almacena los datospersonales.\item idioma: TipoEnum que servira para elegir idioma(0-Castellano, 1-Ingles), 0 por defecto \end{itemize}
