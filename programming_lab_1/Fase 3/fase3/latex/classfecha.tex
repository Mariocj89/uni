\subsection{Descripción detallada}
Clase Fecha, permite tratar fechas reales. Sus atributos son:\begin{itemize}
\item dia: Vector de elementos almacenados\item mes: Numero de elementos actual\item anio: Apuntador al elemento de interes \end{itemize}
